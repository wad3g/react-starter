## Prettier

https://github.com/prettier/prettier-vscode

Configure to format on save: https://github.com/prettier/prettier-vscode#format-on-save

```
{
  "[javascript]": {
    "editor.formatOnSave": true
  },
  "[javascriptreact]": {
    "editor.formatOnSave": true
  }
}
```
